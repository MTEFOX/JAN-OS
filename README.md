# 주의사항
## 이 운영체제는 아사람 혼자 만든 거니 중요한 파일을 보관하시면 제 2의 클로버핏 사태가 날 수 있으니 일상적인 작업만 해주시면 감사하겠습니다.
## -제작자-

# Q. 이 앱은 무슨 앱이냐맨이야
### A. 본명 최잔오이자 활동명 케인이자 은평 독거노인인 자의 팬 운영체제입니다.

# Q. exe 파일은 어떻게 변환하냐맨이야
### A. 터미널로 해당 exe 파일 다운로드 한 경로 들어가서 "wine ./파일명.exe" 하면 됩니다.

# Q. 무슨 프로그램이 있냐맨이야
### A. 파일을 탐색할 수 있는 파일탐색기, 웹 브라우저를 탐색하며 즐기면서 갈 수 있는 뭉탱이폭스, 지하실 아사람 명단을 작성할 수 있는 메모장, 해킹을 조일 수 있는 터미널, 해킹을 막을 수 있는 기본 보안 패키지들이 있습니다.

# 나도 이 코드 일부분 수정해서 컴파일하고 싶썩 (그렇다면, 데비안으로 하셔야 합니다.)

1. Debian 환경에서 실행입니다.
   `sudo apt install live-build`

2. 빌드입니다.
   `chmod +x build.sh`
   `sudo bash build.sh`

3. ISO 생성 완료입니다.
